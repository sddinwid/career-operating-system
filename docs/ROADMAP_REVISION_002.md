# Executive Summary

Revision 002 reflects the evolution of the Career Operating System from a job tracker into a deterministic career operating platform.

The project is no longer centered on application tracking.

Application tracking is now considered infrastructure.

The primary product is now deterministic generation of high-quality career artifacts from structured career knowledge.

The roadmap has therefore been reorganized around that objective.

# Guiding Principles

1. Deterministic before AI.
2. Every derived artifact is immutable.
3. Every pipeline stage is versioned.
4. Every recommendation is explainable.
5. Local-first until commercialization.
6. Repository documentation is the source of truth.

---

# Architectural Direction

Future work should prioritize:

1. Daily usefulness
2. Artifact quality
3. Explainability
4. Deterministic generation
5. Long-term commercial viability

Tracker enhancements become secondary unless they remove significant manual work.
