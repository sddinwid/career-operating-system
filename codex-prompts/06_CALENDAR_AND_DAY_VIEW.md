# Prompt 06 - Calendar and Day Timeline

Implement calendar and day views using the centralized job-search date.

## Requirements

- Month navigation
- Day cells show counts for applications, outreach, responses, interviews, rejections, and offers
- Clicking a day opens a chronological timeline
- Timeline links to applications, contacts, interviews, and documents
- Quick-add activity from day view
- Filters by activity type
- Empty and high-volume days remain usable

## Acceptance criteria

- Imported activity appears on correct job-search dates
- Adjusting an event moves it between day/month views without changing actual recorded time
- Counts match query results
- Tests cover month boundaries
